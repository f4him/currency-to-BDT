# BD Currency Converter

A browser extension that lets you hover over any price on any webpage and instantly see the equivalent in **Bangladeshi Taka (BDT)** at live exchange rates.

No API key required. No setup. Just install and go.

## Features

- Hover over prices like `$1,299`, `€99`, `£49.99`, `₹5,000` to see BDT instantly
- Popup manual converter supporting USD, EUR, GBP, JPY, INR, AUD, CAD, SGD, SAR, AED
- Live rates from [Frankfurter](https://frankfurter.app) — free, open source, no key needed
- Rates cached for 1 hour to stay fast and avoid unnecessary requests
- Works on Amazon, eBay, AliExpress, and most e-commerce sites
- Bangladeshi number formatting (1,00,000 style)
- No data collection, no tracking, no account needed

## Installation (Development)

1. Clone this repo

   ```
   git clone https://github.com/yourusername/bd-currency-converter
   ```

2. Build for your target browser:

   ```
   node build.js chrome
   node build.js firefox
   node build.js edge
   ```

3. Load in browser:
   - **Chrome / Edge**: Go to `chrome://extensions` → Enable Developer Mode → Load unpacked → select folder
   - **Firefox**: Go to `about:debugging` → This Firefox → Load Temporary Add-on → select `manifest.json`

## Project Structure

```
bd-currency-converter/
├── manifests/
│   ├── manifest.chrome.json   — Chrome & Edge manifest
│   └── manifest.firefox.json  — Firefox manifest
├── background.js              — Fetches & caches exchange rates
├── content.js                 — Detects prices on pages, shows tooltip
├── popup.html                 — Toolbar popup UI
├── popup.js                   — Popup converter logic
├── build.js                   — Build script (copies manifest + zips)
└── .gitignore
```

> `manifest.json` in the root is generated by `build.js` and is gitignored.

## Building for stores

```bash
node build.js chrome    # → bd-currency-converter-chrome.zip  (Chrome Web Store)
node build.js firefox   # → bd-currency-converter-firefox.zip (Mozilla Add-ons)
node build.js edge      # → bd-currency-converter-edge.zip    (Edge Add-ons)
```
